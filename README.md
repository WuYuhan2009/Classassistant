# ClassAssistant（班级小助手）

一个面向课堂/班级场景的 Qt 桌面助手，当前版本采用 **悬浮球 + 半圆快捷菜单** 主交互，强调：

- 快速操作（白板 / 考勤 / 息屏 / 随机点名 / AI / 设置）
- 自习课自动息屏与课堂专注场景
- 离线优先 + 可选 AI 增强
- Win7 兼容与 CMake/MSVC 稳定构建

---

## 1. 核心功能

### 1.1 悬浮球与半圆菜单

- 启动默认收起为悬浮球。
- 点击悬浮球展开半圆菜单，再次点击悬浮球可收起。
- 按钮顺序固定：
  1. 白板
  2. 班级考勤
  3. 息屏
  4. 随机点名
  5. AI
  6. 设置
- 菜单按钮与悬浮球中心保持等距，自动避免重叠与越界。
- 点击屏幕其他位置自动收起；无操作超时自动收起。

### 1.2 班级考勤

- 考勤选择（搜索/勾选缺勤/导出）。
- 考勤概览浮窗（应到/实到/请假名单）。
- 强化字体与排版，便于教室远距离查看。

### 1.3 息屏模式（新增）

点击“息屏”后：

- 全屏黑色遮罩置顶显示。
- 右上角保留“考勤概览”。
- 中央偏上显示当前时间（`HH:mm`，大号白色无衬线粗体）。
- 底部四分之一处显示 AI 每日金句：
  - 已填写 API Key：调用硅基流动服务生成
  - 未填写 API Key：提示“填写 API Key 以获取每日金句”
- 右下角按钮：
  - 关机（含二次确认）
  - 退出（退出息屏）

### 1.4 自习课自动息屏（新增）

- 在设置中可添加多节自习课时间段（`HH:mm-HH:mm`）。
- 仅在自习时段内启用“3 分钟无操作自动息屏”。
- 若在自习时段进入息屏，会显示：
  - 自习进度条
  - 剩余时间
- 下课后进度条自动消失，下一节自习重新生效。

### 1.5 AI 助手

- 支持硅基流动 API 在线调用。
- 未配置 API Key 时提供离线兜底建议。

## 2. 三分钟无操作检测实现说明

程序通过全局事件过滤器监听输入事件（鼠标移动/点击、键盘按键、触摸事件），每次输入会刷新“最后操作时间”。

同时使用定时器（10 秒轮询）：

1. 判断当前是否处于已配置的自习时段；
2. 计算 `当前时间 - 最后操作时间`；
3. 当空闲时间 `>= 180 秒` 且本轮空闲未触发过时，自动打开息屏。

这是一种轻量、跨平台兼容性较好的“用户空闲检测”方案。

## 3. 设置项说明（当前）

### 显示与启动

- 悬浮球透明度
- 悬浮球尺寸
- 按钮图标尺寸
- 考勤概览宽度
- 半圆菜单半径
- 无操作自动收起秒数
- 启动时是否收起
- 托盘单击展开
- 启动显示考勤概览

- 点名无重复
- 点名历史条数
- 白板路径
- 分组默认人数
- 队伍名称

### 数据管理

- 名单导入（CSV/TXT）
- 按钮管理（增删改序）

### 安全与离线

- 是否允许打开外链
- AI 配置（API Key / 模型 / 接口地址）

## 4. 数据存储

使用 `QStandardPaths::AppDataLocation`，典型包含：

- `config.json`：所有设置与按钮数据
- `logs/classassistant.log`：运行日志

新增关键配置：

- `floatingBallX`, `floatingBallY`（悬浮球记忆位置）
- `floatingBallSize`
- `radialMenuRadius`
- `menuAutoCollapseSeconds`
- `selfStudyPeriods`（自习课时段）

## 5. 构建

### Linux / macOS

```bash
cmake -S . -B build
cmake --build build
```

### Windows (MSVC)

```bat
cmake -S . -B build -G Ninja -DCMAKE_BUILD_TYPE=Debug
cmake --build build
```

> 若 Qt5 未被 CMake 发现，请设置 `CMAKE_PREFIX_PATH` 或 `Qt5_DIR`。

## 6. 兼容性与注意事项

- 主要面向 Windows 教学机，保留 Win7 兼容编译策略。
- 息屏“关机”按钮会执行系统关机命令，请谨慎使用。
- AI 在线能力依赖网络及有效 API Key。

## 7. 常见问题

### Q1：为什么每日金句没有内容？

- 检查设置里 API Key 是否填写；未填写会显示提示文本。

### Q2：自习课自动息屏没有触发？

- 检查时段格式是否为 `HH:mm-HH:mm`。
- 仅在当前时间位于自习时段内才会启用 3 分钟无操作检测。

### Q3：按钮位置为什么会变化？

- 半圆菜单会根据悬浮球所在左右侧自动调整方向，并在防重叠/防越界条件下计算最佳半径。
