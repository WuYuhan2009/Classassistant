# ClassFlow（班级小助手）

一个面向课堂/班级场景的 Qt 桌面助手，当前版本采用 **悬浮球 + 半圆快捷菜单** 主交互，强调：

- 快速操作（白板 / 考勤 / 息屏 / 随机点名 / AI / 设置）
- 自习课自动息屏与课堂专注场景
- 离线优先 + 可选 AI 增强
- Win7 兼容与 CMake/MSVC 稳定构建

---


## 图标目录与命名规范（Ant Design 风格）

图标统一放在：`assets/icons/`。

当前内置图标命名如下：

- `icon_seewo.svg`（希沃白板）
- `icon_attendance.svg`（班级考勤）
- `icon_screen_off.svg`（息屏）
- `icon_random.svg`（随机点名）
- `icon_ai.svg`（AI 助手）
- `icon_settings.svg`（设置）
- `icon_tray.svg`（托盘图标）
- `bg_cloudbrook.svg`（关于页背景图）

说明：
- 所有内置图标已更换为更通用、简约的 Ant Design 风格 SVG（无背景、无边框），离线可用。
- 系统按钮会优先使用以上规范名称。
- 可通过设置中的按钮管理指定其他图标文件名或绝对路径。

命名建议：
- 系统功能图标统一命名为 `icon_<功能名>.svg`（例如 `icon_ai.svg`）。
- 背景图统一命名为 `bg_<主题名>.svg`（例如 `bg_cloudbrook.svg`）。
- 托盘图标固定命名为 `icon_tray.svg`。

## 1. 核心功能

### 1.1 悬浮球与半圆菜单（Ant Design 规范化布局）

- 启动默认收起为悬浮球。
- 点击悬浮球展开半圆菜单，再次点击悬浮球可收起。
- 按钮顺序固定：
  1. 白板
  2. 班级考勤
  3. 息屏
  4. 随机点名
  5. AI
  6. 设置
- 菜单按钮与悬浮球中心保持等距，自动避免重叠与越界。
- 点击屏幕其他位置自动收起；无操作超时自动收起。
- 功能按钮样式统一为“图标在上、名称在下”，名称单行显示、字号一致、整体居中。

### 1.2 班级考勤

- 考勤选择（搜索/勾选缺勤/导出）。
- 考勤概览浮窗（应到/实到/请假名单）。
- 强化字体与排版，便于教室远距离查看。

### 1.3 息屏模式（ClassFlow 2.0.1 Beta）

点击“息屏”后：

- 全屏黑色遮罩置顶显示。
- 右上角保留“考勤概览”（已修复被黑色遮罩覆盖的问题）。
- 中央偏上显示当前时间（`HH:mm`，大号白色无衬线粗体）。
- 底部四分之一处显示 AI 每日金句：
  - 已填写 API Key：调用硅基流动服务生成
  - 未填写 API Key：提示“填写 API Key 以获取每日金句”
- 右下角按钮：
  - 关机（含二次确认）
  - 退出（退出息屏）


### 1.3.1 每日金句显示规则

- 最多两行。
- 每行最多 15 个汉字。
- AI 生成时会强约束输出长度，并在语义停顿处择优换行，避免机械对齐。

### 1.4 自习课自动息屏（按自习时段触发）

- 在设置中可添加多节自习课时间段（`HH:mm-HH:mm`）。
- 进入自习时段后自动进入息屏（每节时段仅触发一次）。
- 若在自习时段进入息屏，会显示：
  - 自习进度条
  - 剩余时间
- 下课后进度条自动消失，下一节自习重新生效。

### 1.5 AI 助手

- 支持硅基流动 API 在线调用。
- 默认模型：`Qwen/Qwen3-8B`。
- 未配置 API Key 时提供离线兜底建议。

## 2. 自习时段自动息屏实现说明

程序使用定时器（10 秒轮询）检测当前时间是否处于已配置自习时段。

触发规则：

1. 当前时间命中某个 `HH:mm-HH:mm` 自习时段；
2. 同一节自习时段仅自动触发一次息屏；
3. 离开该时段后自动重置，下一节自习可再次触发。

该方案对触控/鼠标输入无依赖，课堂自动化更稳定。

## 4. 数据存储

使用 `QStandardPaths::AppDataLocation`，典型包含：

- `config.json`：所有设置与按钮数据
- `logs/classassistant.log`：运行日志

新增关键配置：

- `floatingBallX`, `floatingBallY`（悬浮球记忆位置）
- `floatingBallSize`
- `radialMenuRadius`
- `menuAutoCollapseSeconds`
- `selfStudyPeriods`（自习课时段，按时段自动触发息屏）

## 5. 构建

### Linux / macOS

```bash
cmake -S . -B build
cmake --build build
```

### Windows (MSVC)

```bat
cmake -S . -B build -G Ninja -DCMAKE_BUILD_TYPE=Debug
cmake --build build
```

> 若 Qt5 未被 CMake 发现，请设置 `CMAKE_PREFIX_PATH` 或 `Qt5_DIR`。

## 8. 常见问题

### Q1：为什么每日金句没有内容？

- 检查设置里 API Key 是否填写；未填写会显示提示文本。

### Q2：自习课自动息屏没有触发？

- 检查时段格式是否为 `HH:mm-HH:mm`。
- 仅在当前时间位于自习时段内才会触发自动息屏。

### Q3：按钮位置为什么会变化？

- 半圆菜单会根据悬浮球所在左右侧自动调整方向，并在防重叠/防越界条件下计算最佳半径。


### Q4：为什么随机点名会自动停下？

- 随机点名已改为 5 秒自动停止并锁定结果。
- 15 分钟内已点到的名字会尽量避开，不重复点名。
